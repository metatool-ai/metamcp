description: "使用内置检查器调试和检查 MCP 服务器和 MetaMCP 端点"
---

**MetaMCP 检查器**是一个内置工具，允许您轻松测试和调试您管理的 MCP 服务器和 MetaMCP 端点。它提供与官方 MCP 检查器类似的功能，提供了一种验证连接和服务器行为的便捷方式。
